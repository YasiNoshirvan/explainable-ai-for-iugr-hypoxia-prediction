# Demo Data

This folder contains only synthetic data examples.

The synthetic data is provided to demonstrate the expected input format of the pipeline and does not represent real patients.

## Expected Columns

| Column | Description |
|---|---|
| `CaseID` | Synthetic case identifier |
| `Status` | `IUGR` or `Normal` |
| `Gestational_Age` | Gestational age |
| `Maternal_Age` | Maternal age |
| `PI_MCA` | Middle cerebral artery pulsatility index |
| `PI_UA` | Umbilical artery pulsatility index |
| `PI_MCA_UA` | MCA/UA Doppler ratio |

## Important Notice

The original clinical dataset is not included in this repository due to data protection, research confidentiality, and future data-publication plans.
