# Demo Data

This folder may contain a small synthetic dataset for demonstrating the expected input format of the pipeline.

The synthetic data does not represent real patients and must not be used for clinical interpretation.

## Expected Columns

| Column | Description |
|---|---|
| `Gestational_Age` | Gestational age |
| `Maternal_Age` | Maternal age |
| `PI_MCA` | Middle cerebral artery pulsatility index |
| `PI_UA` | Umbilical artery pulsatility index |
| `PI_MCA_UA` | MCA/UA Doppler ratio |
| `Status` | `IUGR` or `Normal` |

## Important

The original clinical dataset is not included in this repository.
